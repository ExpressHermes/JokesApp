# Joke App
A random joke generator using nodejs.

## Requirements
* Nodejs
* npm

## Steps to run
 * Fork the repository
 * Clone the repository
 * Run the following commands in the cmd

   ``` 
   $ cd JokesApp 
   $ npm install
   $ node app.js 
   ```
 
* Now head over to http://localhost:3000/ to see it working.
